Git Bash Commands
	$ls			\\Shows files in current directory
	$ls -la			\\list all files in directory 	
	$git init		\\start tracking local codebase 
	$git add -A		\\add to the staging area
	$git status		\\show commit status of files
	$git commit -m "..."	\\commit files to repository, add comment
	$git log		\\shows history of commit
	$touch file.txt		\\create file
